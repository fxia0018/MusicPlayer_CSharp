# 🎵 Jace's Music Player
A terminal-based interactive music player built with **C#** and the **SplashKit SDK**.

## 🌟 Overview
This project is a personalized music player that allows users to listen to my favorite tracks (like *Kokoro no Ame* and *Itsudemo Hohoemi wo*). It features real-time control for pausing, resuming, and switching songs directly from the console.

## 🕹️ Controls
- **`start`**: Open the song selection menu.
- **`1, 2, 3, 4`**: Choose a specific track to play.
- **`p` / `P`**: Toggle between Pause and Resume.
- **`q` / `quit`**: Stop music and exit the player.
